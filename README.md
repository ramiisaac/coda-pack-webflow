# Webflow Coda Pack

Integrate Webflow's Data API seamlessly into your Coda docs with this pack. Fetch and sync your Webflow sites and collections, and automate your workflows with custom formulas.

## Features

- **Fetch Sites:** Retrieve details of your Webflow sites.
- **Fetch Collections:** Get all collections within a specified site.
- **Sync Tables:** Automatically sync Webflow sites and collections into your Coda tables.
- **Custom Formulas:** Use formulas to fetch and manipulate Webflow data.

## Installation

1. Open your Coda doc.
2. Click on the `+` button to add a new Pack.
3. Search for `Webflow` and select the pack.
4. Authenticate with your Webflow account when prompted.

## Usage

Refer to the [examples](./examples.md) for detailed instructions on using the pack's features.

### Fetching a Webflow Site

Use the `GetWebflowSite` formula to fetch details of a specific site.
